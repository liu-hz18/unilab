# Unilab Deployer

## powered by Vue2 & Golang

## Project build
```
sudo make build
```

### Project deploy
```
sudo make deploy 
```

### Stop deploy
```
sudo make stop
```

### Browse
```
type `https://lab.cs.tsinghua.edu.cn/unilab/` to find your course.
type `https://lab.cs.tsinghua.edu.cn/unilab/report.html` to view frontend pages report.
```
